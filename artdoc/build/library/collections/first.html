first

return the first item of the given collection

Parameters

first  collection :string :block

Attributes

n :integer get first items

Returns

  • :any

Examples

print first "this is some text"       ; t
		print first ["one" "two" "three"]     ; one
		
		print first.n:2 ["one" "two" "three"] ; one two
